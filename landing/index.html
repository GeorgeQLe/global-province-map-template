How it works

Scaffold → graph → politics → export

Geometry and historical politics evolve on different cadences. Official eras start as curated ownership overlays; period shapes arrive only where modern outlines fail the sniff test.

  1. 01

    Open sources

    Natural Earth, geoBoundaries, optional population rasters — manifests with checksums and license metadata.

  2. 02

    Provinces & seas

    Deterministic IDs, coastal and ocean sea zones, ports and strait shortcuts, topology QA gates.

  3. 03

    Scenario politics

    Owner, controller, cores, and claims layered via baseline projection plus region and province overrides.

  4. 04

    Dual export

    Game packs for modders and studios; atlas packs for choropleths, legends, and SaaS ingestion.

Historical accuracy bar

Paradox-adjacent eyes are welcome

Anachronisms destroy trust. Official eras aim for a strong balance — not pure archival GIS on day one, and not “modern ISO with period skins.” Labels are honest about what shipped.

Quality tier Meaning Status
scaffold-baseline Modern open-geodata shapes; politics project modern parents or coarse demos. Default geometry today
curated-politics Human-reviewed owner/controller trees for supported start dates. official-1836 · official-1444
period-geometry Era-aware boundaries where modern outlines break the political story. M15+ roadmap

We do not claim

  • Paradox-grade historical accuracy worldwide
  • Period-correct province geometry (yet)
  • Legal maritime boundaries (sea zones are gameplay abstractions)
  • Complete global politics outside priority theaters

Data policy

Permissive core, isolated options

Public packs refuse restricted and share-alike contamination. Optional paths stay behind policy walls with isolation notices in the attribution pack.

Public-safe core

Natural Earth (PD), geoBoundaries (CC BY), optional GHSL / WorldPop when lineage is present.

Isolated

OpenStreetMap / ODbL may power optional experiments but does not enter public redistributable packs.

Excluded

GADM and other restricted sources are catalogued as blocked for default and public builds.

Get started

Clone, generate, package

Requires Python 3.11+, uv, and downloaded open source artifacts. Full docs live in the repository README and docs/ milestone notes. 

git clone https://github.com/GeorgeQLe/global-province-map-template.git
cd global-province-map-template
uv sync
uv run gpm sources download --execute --profile modern-small
uv run gpm build provinces --profile modern-small
uv run gpm release beta --sample-we
