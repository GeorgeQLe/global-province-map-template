# M13 Second Curated Official Scenario (1444)

M13 ships the second **official era program**: `official-1444`, a
**curated-politics** ownership overlay for the EU-leaning **1444-11-11** start
date.

Geometry is still the modern Natural Earth scaffold. Politics are human-authored
country/region rules (and optional province overrides) over that scaffold—not
period-correct borders. See the historical accuracy bar in [ROADMAP.md](../ROADMAP.md).

## What shipped

| Artifact | Role |
| --- | --- |
| `configs/scenarios/official-1444.json` | Official era definition (`quality_tier=curated-politics`) |
| `configs/scenarios/golden/official-1444.json` | Golden tag floors for CI (`min_owner_counts`) |
| `configs/recipes/official-1444-curated.json` | Reproducible build/QA/export recipe (`eu-like`) |
| Accuracy labeling | `official-1444` recognized in `gpm.release.quality.accuracy_label` |

Compared to `demo-1444` (pedagogical coarse remaps, **scaffold-baseline**) and
`official-1836` (Victoria 1836), `official-1444` is labeled **official** with
**Europe-first elevated depth** plus global major-power tags.

## Commands

```bash
uv run gpm scenario list
uv run gpm scenario validate --scenario official-1444
uv run gpm scenario build --scenario official-1444 --profile eu-like
uv run gpm qa scenario --scenario official-1444 --profile eu-like
uv run gpm review --scenario official-1444
uv run gpm export pack --scenario official-1444 --profile eu-like
uv run gpm export atlas --scenario official-1444 --profile eu-like
```

`gpm qa scenario` **auto-loads** `configs/scenarios/golden/<scenario-id>.json`
when present. Pass `--golden` to override.

## Curation approach

Assignment precedence is unchanged from M8 (later layers win field-by-field):

1. **Baseline** — modern `parent_country_id` → owner/controller  
2. **Country rules** — global major-power remaps (England from GBR, Castile from
   ESP, Burgundian Low Countries, Muscovy from RUS, Ming from CHN, Mamluks from
   EGY/Levant, pre-Columbian Americas as `NAH`, etc.)  
3. **Region rules** — elevated theaters:  
   - **Western Europe:** Burgundy in eastern France + Low Countries, Brittany,
     Provence, English Calais; Castile / Aragon / Navarre / Granada on Iberia  
   - **British Isles:** Scotland and Ulster peeled from UK scaffold; Wales under
     England with `WLS` cores  
   - **HRE:** Bavaria, Saxony, Brandenburg, Palatinate, free cities, Danish
     Schleswig-Holstein over the German scaffold  
   - **Italy:** Naples, Papal States, Florence, Siena, Lucca, Milan, Mantua,
     Venice, Genoa, Savoy, Ferrara/Modena, Aragonese Sardinia, Tyrol Habsburg  
   - **East Europe:** Teutonic Baltic, Novgorod NW Russia, Transylvania Hungarian,
     Wallachia / Moldavia split of modern Romania, Crimean residual  
   - **Mediterranean:** Byzantine Constantinople remnant, Morea if present  
   - **India majors:** Vijayanagar south, Gujarat, Bengal, Bahmani Deccan  
4. **Province overrides** — empty in the shipped definition; curators add via
   `gpm review --scenario official-1444`

Honest limits:

- Residual modern ISO baselines and `NAH` / `UNK` tags remain outside priority
  theaters.
- Modern admin outlines are **not** 1444 borders (M15+ period geometry)—especially
  painful for HRE micro-states and French appanages.
- Not Paradox-grade completeness; golden floors enforce major-tag presence, not
  every frontier or OPM.
- `demo-1444` stays available for tooling demos and must not be marketed as the
  official era.

## Golden checks

`configs/scenarios/golden/official-1444.json` sets `min_owner_counts` for major
1444 tags (e.g. ENG, FRA, BUR, CAS, ARA, HAB, MOS, TUR, MNG, VEN, NAP, PAP, BYZ,
GRA, TEU). Province-level `province_owners` can be added later with stable
`province_id`s for famous borders.

## Quality labeling

`gpm.release.quality.accuracy_label` recognizes `official-1444` and describes it
as curated-politics. Geometry remains `scaffold-baseline` until era-aware
geometry (M15+). Public alpha packaging (M9) still defaults both tiers to
scaffold-baseline unless a release explicitly claims higher politics fidelity.

## Why 1444 (not 1936)

Roadmap Phase 11 lists both 1444 and 1936 as second-era candidates. M13 chose
**1444** because:

1. `demo-1444` already exercised the era tooling path.
2. `eu-like` profile alignment matches the Europe-first curation bar.
3. 1836 (M12) + 1444 (M13) cover the two densest strategy-game start dates
   before interwar **1936** (HOI-leaning) in a later milestone.
