# M12 First Curated Official Scenario (1836)

M12 ships the first **official era program**: `official-1836`, a
**curated-politics** ownership overlay for the Victoria-leaning **1836** start
date.

Geometry is still the modern Natural Earth scaffold. Politics are human-authored
country/region rules (and optional province overrides) over that scaffold—not
period-correct borders. See the historical accuracy bar in [ROADMAP.md](../ROADMAP.md).

## What shipped

| Artifact | Role |
| --- | --- |
| `configs/scenarios/official-1836.json` | Official era definition (`quality_tier=curated-politics`) |
| `configs/scenarios/golden/official-1836.json` | Golden tag floors for CI (`min_owner_counts`) |
| Scenario schema fields | `quality_tier`, `official_era`, `recommended_profile`, `priority_theaters` |
| Null-safe ID matching | Region/country ids strip shapefile NUL padding so rules match |

Compared to `demo-1444` (pedagogical coarse remaps) and `modern-baseline`
(scaffold projection only), `official-1836` is labeled **official** and aims for
global major-power tags plus **elevated depth** in priority theaters.

## Commands

```bash
uv run gpm scenario list
uv run gpm scenario validate --scenario official-1836
uv run gpm scenario build --scenario official-1836 --profile victoria-like
uv run gpm qa scenario --scenario official-1836 --profile victoria-like
uv run gpm review --scenario official-1836
uv run gpm export pack --scenario official-1836 --profile victoria-like
uv run gpm export atlas --scenario official-1836 --profile victoria-like
```

`gpm qa scenario` **auto-loads** `configs/scenarios/golden/<scenario-id>.json`
when present. Pass `--golden` to override.

## Curation approach

Assignment precedence is unchanged from M8 (later layers win field-by-field):

1. **Baseline** — modern `parent_country_id` → owner/controller  
2. **Country rules** — global major-power remaps (UK+Ireland, French Algeria,
   Qing China, Dutch East Indies, British settler colonies, etc.)  
3. **Region rules** — elevated theaters:  
   - **Europe:** German Confederation members (Bavaria, Saxony, Hanover, …),
     Italian states (Sardinia-Piedmont, Papal States, Two Sicilies, Tuscany, …),
     Polish partitions, Transylvania under Austria  
   - **North America:** Republic of Texas, Mexican North, disputed Oregon Country,
     Russian Alaska, Kingdom of Hawaii  
   - **Colonial:** British/French/Dutch/Spanish empire defaults for key ISO parents  
4. **Province overrides** — empty in the shipped definition; curators add via
   `gpm review --scenario official-1836`

Honest limits:

- Residual modern ISO baselines remain outside priority theaters (~baseline-only
  provinces after resolve).
- Modern admin outlines are **not** 1836 borders (M15+ period geometry).
- Not Paradox-grade completeness; golden floors enforce major-tag presence, not
  every frontier.

## Golden checks

`configs/scenarios/golden/official-1836.json` sets `min_owner_counts` for major
1836 tags (e.g. GBR, FRA, PRU, AUS, RUS, USA, MEX, TEX, BAV, SAR, SIC, PAP).
Province-level `province_owners` can be added later with stable
`province_id`s for famous borders.

## Quality labeling

`gpm.release.quality.accuracy_label` recognizes `official-1836` and describes it
as curated-politics. Geometry remains `scaffold-baseline` until era-aware
geometry (M15+). Public alpha packaging (M9) still defaults both tiers to
scaffold-baseline unless a release explicitly claims higher politics fidelity.
